Standard Installation / 标准安装

Service Info / 服务信息

Install home fixtures (<5kg) on walls/ceilings. NO ELECTRICITY or WATER involved.

在墙面/天花板，安装重量不超出5公斤的任何东西.安装过程不牵涉到水和电.

Examples / 例子:

Wall Paintings （壁画）, Hooks （挂钩）, Battery Clocks （电池钟）, Clothes Hangers （衣架）, Key Holders （钥匙扣架）, Bathroom Shelves （浴室架）or Toilet Roll Holders （厕纸吊勾等等） etc.

Minimum booking / 最短预约: 1 hour / 1小时

Client can install any number of items. Overtime will be based on 30-min block charges.

不限制安装数量，如超时，以半小时计算。

Query Call / 询问电话: +65 9833 0259 or +65 8288 1445.

Electrical Installation / 家电安装

Service Info / 服务信息

Install home fixtures (< 8kg) on walls/ceilings. NO RUNNING OF ELECTRICAL CABLE FOR NOW.

少于8公斤的家电墙壁/天花板安装。

Examples / 例子:

Wall or Ceiling Fans （风扇）, Water Heaters （热水器）, Lights （灯饰）, Door Bells （门铃）, Change Bulbs （换灯泡）, Replace faulty Electrical Sockets （更换电插头等等） etc.

Minimum booking / 最短预约: 1 hour / 1小时

Query Contact / 询问电话: +65 9833 0259 or +65 8288 1445.
