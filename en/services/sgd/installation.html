Standard Installation / 标准安装

Service Info / 服务信息

Install home fixtures (< 5kg) on walls/ceilings. No water, electricity or human safety involved.

在墙面/天花板，安装重量不超出5公斤的任何东西.安装过程不涉及水，电和人身安全.

Examples / 例子:
  • Wall Paintings / 壁画.
  • Hooks / 挂钩.
  • Battery Clocks / 电池钟.
  • Clothes Hangers / 衣架.
  • Key Holders / 钥匙扣架.
  • Bathroom Shelves / 浴室架.
  • Toilet Roll Holders / 厕纸吊勾.
  • Dismantle or install curtain / 拆或安装窗帘布 (< 3m) etc.

Minimum booking / 最短预约: 1hr（小时）

Client can install any number of items. Service extension will be charged at 30-min blocks.

Blooms charges by actual service duration (min 1 hr). If tasks can be completed faster, we charge the shorter duration, also in 30-min blocks.

不限制物件数量。如客户临时要加安装物件，加时收费便以30分钟为计算单位。

家盟 (Blooms) 会以实际时段（最少1小时）收费。如安装服务预订提早完成，会以30分钟为计算单位收费。

Technical Installation / 技术安装

Service Info / 服务信息

Installation involving / 安装涉及到:

  • Electricity / 电
  • Water or fluid / 水
  • Direct human safety / 人身安全
  • Structural safety / 结构安全
  • Items at heights / 高处物件 (> 3m)
  • Heavy items / 重物件 (> 8kg)
  • Glass / 玻璃
  • Assembly tasks / 组装物件
  • Signal, Radio Frequency, Internet, LAN cabling / 网路，讯号电缆等
Examples / 例子:
  • Wall or Ceiling Fans / 风扇
  • Water Heaters / 热水器
  • Lights / 灯饰
  • Door Bells / 门铃
  • High curtain systems / 高窗帘
  • Bulb Change / 换灯泡
  • Replace faulty electrical sockets / 更换电插头等等

Minimum booking / 最短预约: 1hr（小时）

New Parts Picking Charge / 代购费

All services exclude new part purchase. If Blooms is to replace any faulty parts, a material picking charge of SGD30 will be incurred. For the item purchased, customer will pay the market price (at no markup) for it.

安装服务不包括替代件（如灯管，热水器，插头等）。如要家盟 (Blooms) 代购，公司会另加 SGD30 代购费。
