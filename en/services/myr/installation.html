Standard Installation 标准安装

Service Info

Install home fixtures (<5kg) on walls/ceilings. NO ELECTRICITY or WATER INVOLVED.

Examples / 例子:

Wall Paintings （壁画）, Hooks （挂钩）, Battery Clocks （电池钟）, Clothes Hangers （衣吊）, Key Holders （钥匙扣）, Bathroom Shelves （浴室架）or Toilet Roll Holders （厕纸调勾等等） etc.

Minimum booking / 最短预约: 1 hour / 1小时

Client can install any number of items. Overtime will be based on 30-min block charges. 不限制安装数量，如超时，已半小时计算。

Query Call / 询问电话: +60 10 883 0800.

Electrical Installation 家电安装

Service Info

Install home fixtures (< 8kg) on walls/ceilings. NO RUNNING OF ELECTRICAL CABLE FOR NOW.

少于8公斤的家电墙壁/天花板安装。

Examples:

Wall or Ceiling Fans （风扇）, Water Heaters （热水器）, Lights （灯饰）, Door Bells （门铃）, Change Bulbs （换灯泡）, Replace faulty Electrical Sockets （更换电插等等） etc.

Minimum booking / 最短预约: 1 hour / 1小时

Query Contact / 询问电话: +60 10 883 0800.
