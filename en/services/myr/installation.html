Standard Installation / 标准安装

Service Info / 服务信息

Install home fixtures (<5kg) on walls/ceilings. No water, electricity or human safety involved.

在墙面/天花板，安装重量不超出5公斤的任何东西.安装过程不牵涉水，电和各人安全.

Examples / 例子:
  • Wall Paintings / 壁画
  • Hooks / 挂钩
  • Battery Clocks / 电池钟
  • Clothes Hangers / 衣架
  • Key Holders / 钥匙扣架
  • Bathroom Shelves / 浴室架
  • Toilet Roll Holders / 厕纸吊勾
  • Anything that one crew can manage / 1人能胜任的任何物件

Minimum booking / 最短预约: 1hr (小时).

Client can install any number of items. Service extension will be charged on 30-min blocks.

不限制安装数量，如超时，以30分钟块计算。

Technical Installation / 技术安装

Service Info / 服务信息

Installation involving / 安装牵涉到:

  • Electricity / 电
  • Water or fluid / 水
  • Direct human safety / 人的安全
  • Structural safety / 结构安全
  • Items at heights / 高位置物件 (>3m)
  • Heavy items / 重物件 (>8kg)
  • Glass / 玻璃
  • Assembly tasks / 组装物件
  • Signal, Radio Frequency, Internet, LAN cabling / 网路，讯号电缆等
Examples / 例子:
  • Wall or Ceiling Fans / 风扇
  • Water Heaters / 热水器
  • Lights / 灯饰
  • Door Bells / 门铃
  • High curtain systems / 高窗帘
  • Bulb CHange / 换灯泡
  • Replace faulty electrical sockets / 更换电插头等等

Minimum booking / 最短预约: 1hr / 1小时.

New Parts Picking Charge / 代购费

All services exclude new part costs. If Blooms is to supply any faulty parts, we will charge market price for the replacement. In such case, material picking charge will be incurred.

安装服务不包括替代件（如灯管，热水器，插头等）。如要Blooms代购，公司会另加收费。
